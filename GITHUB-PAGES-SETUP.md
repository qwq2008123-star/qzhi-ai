# GitHub Pages 部署指南

## 第一步：进入仓库 Settings

1. 打开浏览器，访问：https://github.com/qwq2008123-star/qzhi-ai

2. 点击顶部的 **"Settings"** 标签（不是左侧菜单，是在页面顶部的标签栏）

   📍 位置：在 Code、Issues、Pull requests 等标签的右边

## 第二步：找到 Pages 设置

在 Settings 页面：

1. 滚动到页面左侧菜单
2. 找到并点击 **"Pages"** 选项

   📍 位置：在 "Secrets and variables" 下面，或者在 "Code and automation" 下面

## 第三步：配置 GitHub Pages

在 Pages 设置页面：

1. 找到 **"Source"** 部分（第一个下拉菜单）
2. 选择 **"Deploy from a branch"**

3. 第二个下拉菜单（Branch）：
   - 选择 **"main"**

4. 第三个下拉菜单（文件夹）：
   - 选择 **"/ (root)"**

5. 点击右侧的 **"Save"** 按钮

## 第四步：等待部署

保存后，等待 2-3 分钟。

成功标志：页面顶部会出现绿色横幅：
**"Your site is published at https://qwq2008123-star.github.io/qzhi-ai"**

## 常见问题

### Q: 找不到 Pages 选项？
A: Pages 选项在 Settings 左侧菜单中，可能需要向下滚动才能看到

### Q: 保存后还是 404？
A: GitHub 需要 2-5 分钟来部署网站，请耐心等待

### Q: 显示 "Pages needs to be verified"?
A: 按照 GitHub 的提示验证您的邮箱即可
